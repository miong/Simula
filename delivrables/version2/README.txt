Utilisation de la version 2.0 de Simula

Exécution du programme

Vous pouvez utiliser le fichier jar deja compilé (recommandé) grâce à la commande :

# scala Simula_V2.0.jar

/!\ L'interface text du jar présente des bugs du a l'encodage.


Sinon les fichers du projet se trouvent dans le dossier source_code.
Il suffit de copier l'intégralité de ce dossier et d'utiliser SBT
depuis la destination de la copie.
# cd destination
# sbt run

Il existe d'autres manières de lancer le projet mais celles-ci ne sont pas conseillées
dues à leurs complexités ( spécification de classpath ect. ).
