Utilisation de la version 2.0 de Simula

Execution du programme

Vous pouvez utilisé le fichier jar deja compilé (recommander) grace à la commande :

# scala Simula_V2.0.jar

Sinon les fichers du projet se trouve dans le dossier source_code.
Il suffit de copier l'intégralité de se dossier et d'utiliser SBT
depuis la destination de la copie.
# cd destination
# sbt run

Il existe d'autres manières de lancer le projets mais celle-ci ne sont pas conseillées
due a leurs complexitées ( spécification de classpath ect. ).
